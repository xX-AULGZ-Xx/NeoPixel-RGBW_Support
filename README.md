# ส่วนเสริม NeoPixel RGBW สำหรับ microBlock (NeoPixel RGBW Extension for microBlock)

ส่วนเสริม (Extension) สำหรับใช้งานบนโปรแกรม microBlock สำหรับควบคุมไฟเส้น LED NeoPixel รุ่น WS2812B (RGB) และ SK6812 (RGBW) โดยใช้บอร์ดไมโครคอนโทรลเลอร์ ESP32

## คุณสมบัติ (Features)
- รองรับการทำงานทั้งโหมด **RGB** ปกติ (3 ไบต์ต่อพิกเซล) และโหมด **RGBW** (4 ไบต์ต่อพิกเซล)
- สามารถระบุ Pin และจำนวนหลอดไฟ LED ในหน้าต่างตั้งค่าตั้งต้นได้
- มีบล็อกที่ใช้งานง่าย ทั้งสำหรับการสั่งเปลี่ยนสีทีละดวง หรือสั่งให้เป็นสีเดียวกันทั้งเส้น
- มีบล็อกไฟแบบสายรุ้ง (Rainbow) ให้เลือกใช้งานทันที
- สามารถปรับระดับความสว่างหลักได้

## บล็อกที่สามารถใช้งานได้ (Blocks Available)

- **Setup Block:** บล็อกสำหรับเริ่มต้นและตั้งค่าการทำงาน (ระบุ Pin, จำนวนหลอด LED, และเลือกว่าเป็นโหมดการทำงาน `RGB` หรือ `RGBW`)
- **Set Color (RGB/RGBW):** บล็อกตั้งค่าให้หลอด LED ดวงใดดวงหนึ่งสว่างเป็นสีที่ต้องการ
- **Fill Color (RGB/RGBW):** บล็อกตั้งค่าให้หลอดไฟสว่างทั้งหมดด้วยดวงสีที่ต้องการ
- **Show:** คำสั่งอัปเดตและแสดงผลสีที่ตั้งค่าเอาไว้ไปยังหลอด LED (ต้องเรียกใช้ทุกครั้งเพื่อให้ไฟเปลี่ยนสีจริง)
- **Clear:** สั่งปิดหลอดไฟทั้งหมด
- **Rainbow:** แสดงภาพไฟขยับเป็นแสงสีรุ้งหมุนวน
- **Set Brightness:** ปรับระดับความสว่างทั้งหมด (0-100%)

## การติดตั้งและการใช้งาน (Installation)
สามารถคัดลอกลิงก์ Repository ของ GitHub โครงการนี้ นำไปใส่ที่หน้า Extension Manager ภายในโปรแกรม IDE microBlock เพื่อติดตั้งส่วนเสริมและใช้งานได้ทันที

## ความเข้ากันได้ของระบบ (Compatibility)
- ทดสอบและใช้งานร่วมกับ ESP32 บนฐานระบบของ MicroPython
- ตัวโค้ดรองรับการทำงานทั้งในแบบ `bpp=3` (RGB) และ `bpp=4` (RGBW) ซึ่งจะป้องกันและแก้ไขปัญหา `IndexError: tuple index out of range` ได้แล้วเมื่อใช้กับไฟเส้นรุ่น RGBW
